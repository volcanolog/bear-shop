Запуск
1) скачать себе репозиторий
2) установить зависимости
3) в папке frontend выполнить npm start
4) в папке backend выполнить npm run dev

Практическая работа №1: CSS-препроцессоры

Цель: Изучение инструментов SASS/LESS для расширения возможностей CSS.

    Что добавлено:

        Реализована карточка товара (название, описание, фото).

        Использованы переменные для управления цветовой схемой и размерами.

        Созданы миксины для переиспользования стилей (например, для оформления кнопок).

        Применена вложенная структура селекторов для улучшения читаемости кода.

Практическая работа №2: Сервер на Node.js + Express

Цель: Разработка серверной части и реализация базовой логики API.

    Что добавлено:

        Инициализирован проект на Node.js и установлен фреймворк Express.

        Реализован полный цикл CRUD для сущности «Товар»:

            GET /products — получение всех товаров.

            GET /products/:id — поиск товара по ID.

            POST /products — добавление нового товара.

            PATCH /products/:id — частичное обновление данных.

            DELETE /products/:id — удаление товара.
